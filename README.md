# rubiks-cube-scramble-cli

A Node CLI that gives you Rubik's cube scrambles.

# Getting started

``` shell
npm install --global rubiks-cube-scramble-cli

getScramble
# Get Rubik's cube scramble

getScramble --number=5
# Get 5 Rubik's cube scrambles

getScramble -n=5
# Get 5 Rubik's cube scrambles
```
